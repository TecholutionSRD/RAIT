Simulation Tools

Simulation Mode

▼

Endpoint Information

URL: /simulate

Method: POST

Description: Opens a new browser tab showing a 3D simulation of the robot executing tasks.

Request Format

{
        "enable_simulation": true
    }

Parameters

Parameter Type Description Required
enable_simulation boolean Enable or disable simulation mode Yes

Response Format

{
        "status": "success",
        "message": "Simulation started successfully"
    }

AI Tools

Gemini Inference

▼

Endpoint Information

URL: /gemini_inference

Method: POST

Description: Performs object detection using Google's Gemini Model on the provided image.

Request Format

{
    "image": "base64_encoded_image",
    "target_classes": ["class1", "class2"]
}

Parameters

Parameter Type Description Required
image string Base64 encoded image data Yes
target_classes array Array of class names to detect Yes

Response Format

{
    "detected_objects": {
        "object_name": [xmin, ymin, xmax, ymax]
    }
}

Status Codes

  • 200 Successful detection
  • 400 Invalid input format
  • 500 Server error

Video Recorder

▼

Endpoint Information

URL: /video_recorder

Method: POST

Description: Records an 8-second video at 30 FPS and saves frames at 2 FPS, while also capturing initial RGB and depth images.

Request Format

{
    "num_recordings": 1,
    "action_name": "pouring",
    "objects": ["red soda can", "white cup"]
}

Parameters

Parameter Type Description Required
num_recordings integer Number of video recordings to capture Yes
action_name string Name of the action being recorded Yes
objects array Array of object names involved in the action Yes

Response Format

{
    "status": "success",
    "message": "Video and frames saved successfully."
}

Status Codes

  • 200 Recording successful
  • 400 Invalid parameters
  • 500 Recording error
